
# AnimatedTouch Component

Wrap mobile UI elements with this to give them lively touch/hover animations. Perfect for buttons and cards.

## Example usage

```tsx
import AnimatedTouch from "@/components/ui/AnimatedTouch";

<AnimatedTouch>
  <button className="puzzle-button w-full my-2">Start Level!</button>
</AnimatedTouch>
```

This adds scale/tap/bounce/wiggle animation for an eye-catching, mobile-optimized feel.
